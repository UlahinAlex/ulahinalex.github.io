Fudo Security

Fudo Security – ведущий производитель решений по управлению привилегированным доступом, один из немногих в мире, кто занимает высший эшелон в индустрии. Список заказчиков по всему миру включает в себя финансовые институты, энергетические компании, а также общественные и государственные организации. Дистрибьюторская сеть охватывает 80 партнеров в 30 странах, со штаб-квартирой в Ньюарке, Калифорния.

Fudo PAM

Привилегированные учетные записи обычно имеют неограниченный доступ к жизненно важным ресурсам IT: главным серверам, сетевому оборудованию или к рабочим станциям пользователей.

Недостаток контроля за привилегированными пользователями является прямой угрозой целостности и безопасности ваших данных. Fudo PAM является наиболее совершенным решением по управлению привилегированным доступом, которое состоит из следующих модулей:

- Secret Manager (политики управления паролями привилегированных учетных записей);

- Efficiency Analyzer (комплексный инструмент для анализа производительности);

- Privileged Session Monitoring (управление и мониторинг привилегированных сессий);

- Application to Application Password Manager (совместно с Secret Manager обеспечивает работу приложений с паролями).
